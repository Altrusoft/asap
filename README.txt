To get started in eclipse:

1. Checkout in eclipse workspace.
2. New Java Project
3. In settings for Java Build Path
   [AddVariable]
   [Configure Variables]
   [New...]
   Name: ALFRESCO_SDK_42C
   Path: <path-to-your-sdk>
